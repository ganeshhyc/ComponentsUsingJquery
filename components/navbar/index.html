Navbar Components using Jquery

  • .nav and #mynav are the class and id need to be declared to initialize the navbar.
  • .active for the default selected element.
  • .icon for the responsive behaviour when used in a compact device.

Code

<div class="nav" id="mynav">
        <a href="#home" class="home active">Home</a>
        <a href="#login" class="login">Login</a>
        <a href="#contact class="contact">Contact</a>
        <a href="#about" class="about">About</a>
        <a href="javascript:void(0);"class="icon">
               <img src="images/burger.png" width="17px">
        </a>
</div>
